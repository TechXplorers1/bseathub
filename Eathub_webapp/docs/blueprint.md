# **App Name**: SwiftDash

## Core Features:

- Restaurant Listings: Display a list of restaurants available for delivery based on user's location, replicating DoorDash's restaurant browsing experience.
- Search and Filtering: Implement robust search and filter options, enabling users to find restaurants and specific dishes quickly, similar to DoorDash's search functionality.
- Menu Display: Showcase restaurant menus with detailed item descriptions, images, and pricing, mirroring the menu presentation on DoorDash.
- Order Placement: Enable users to place orders, customize items, and specify delivery instructions, matching DoorDash's order placement process.
- Delivery Tracking: Provide real-time tracking of order status and delivery progress on a map, similar to DoorDash's tracking feature.
- Recommendations: Recommend meals based on customer preference and search history by acting as an orchestration tool on customer's data, enabling a customized user experience.

## Style Guidelines:

- Primary color: Vibrant orange (#FF6B00) to convey energy and appetite.
- Background color: Light gray (#F5F5F5) to provide a clean and neutral backdrop.
- Accent color: Teal (#008080) for highlighting key interactive elements.
- Body and headline font: 'Inter' (sans-serif) for a modern and readable interface.
- Use recognizable food-related icons, with orange as a highlight.
- Emulate DoorDash's grid-based layout for restaurant listings, with clear, card-based designs.
- Implement subtle animations for transitions and loading states to improve user experience.
